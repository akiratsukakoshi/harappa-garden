## 2. 抽出・作成のルール
* **時間軸の統合:** 会話の時系列順ではなく、「トピックごと」に情報を集約すること。
* **ニーズの強弱判定:** 顧客の熱量に基づき、重要度を3段階（★〜★★★）で判定すること。
* **事実と想いの分離:**
    * **[Fact/確定]:** 予算、決裁フロー、数値など確定事実。
    * **[Hope/想い]:** 「〜だったらいいな」という主観や理想。
    * **[Guess/推測]:** 確信がない仮説段階の発言。
