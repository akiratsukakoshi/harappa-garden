# HARAPPA Management Garden (HMG)

原っぱ大学(株式会社HARAPPA)の経営業務を **AIエージェント中心** で運用する統合プラットフォーム。

塚越さんは **庭師(Gardener)**、AIエージェント群は **自律的に育つ生態系**。HMC(Harappa Management Cockpit)で培った業務スキルを基盤に、エージェントが起点となって業務を進める構造へ進化させていく。
