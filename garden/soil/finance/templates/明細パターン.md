# 明細パターン型ライブラリ(見積 d / 請求 f の標準明細)

過去の発行物から抽出した **案件種別ごとの標準明細**。見積/請求を起こすとき、
ここの品名・標準単価・注記を下敷きにして案件台帳の実額で埋める(ゼロから書かない)。

> **使い方**: 案件の型(研修連鎖 / 単発イベント運営 / 継続運営)を選び、該当パターンの
> 明細行を `line_items` の素体にする → 案件台帳の finance_links で単価・数量を実額化 →
> [issue_document.py](../../../services/finance/issue_document.py) で雛形 .xlsx に差し込む。
>
> 単価は **目安**(案件規模で動く)。注記(※…)は品名に含める運用(雛形の品名セルは1セル)。

---

## パターン A: 研修・単発イベント運営型(MTI / 三菱UFJ / ゼンリン)

合宿・新人研修・単発イベント。**1案件 = 1発行**。明細の骨格:

| 品名(注記込み) | 数量の意味 | 標準単価(目安) |
|---|---|---|
| 全体企画費　※企画設計、プランニング | 一式=1 | 300,000〜600,000 |
| 運営・プロデュース費　※当日運営 | 運営日数 | 150,000〜200,000 |
| 運営サポートスタッフ　単位：人日 ※安全管理含む | 人日 | 50,000 |
| 資材・備品代(／備品レンタル ※調理器具・工具) | 一式=1 | 50,000〜120,000 |
| 資材準備及び手配 ※事前手配・輸送・運搬・廃棄含む | 一式=1 | 100,000〜150,000 |
| 交通費・諸経費 ※レンタカー・前泊・下見交通費含む | 一式=1 | 100,000〜150,000 |

**食事を立てる場合**(合宿系・単価×人数×回):

| 品名 | 数量 | 単価(目安) |
|---|---|---|
| ランチお弁当(N回) | 人数×回 | 2,000 |
| 夕ご飯代 ※飲み物代別途 | 人数×回 | 1,500〜3,000 |
| 朝食代 | 人数×回 | 700〜1,000 |

**宿泊・会場を立てる場合**(実費ベース):

| 品名 | 数量 | 単価 |
|---|---|---|
| シェアハウス／宿泊費 | 泊数 | 実費 |
| 体育館／研修室利用料 | 枠数 | 実費 |

**オプション行**(案件次第):
- 事前オンラインゲスト(20,000 目安)
- LINEオープンチャット運用／集客(50,000 目安)
- 会場費・宿泊費・食費一式　※別途お見積りの通り(一括計上する様式)

## パターン B: 継続運営型(パナHM みずき台 / 三井 / 京急桐畑)

月次・継続でコミュニティ運営。**回数(月数)を数量に乗せる**のが特徴(パナHM見積は各行 ×6)。

| 品名 | 数量の意味 | 単価(目安) |
|---|---|---|
| 全体企画費　※企画設計、プランニング | 一式=1 | 200,000 |
| 運営・プロデュース費　※当日運営 | 開催回数 | 170,000 |
| 運営サポートスタッフ　単位：人日 ※安全管理含む | 人日(回数) | 50,000 |
| 資材・備品代 | 回数 | 20,000 |
| 移動交通費 ※遠隔地は新幹線移動 | 回数 | 50,000 |
| 諸経費(消耗品費・資材配送料・保険代など) | 回数 | 35,000 |
| LINEオープンチャット運用／集客 | 回数 | 50,000 |

## 内部原価試算(★クライアント非開示・発行物には載せない)

過去の MTI 見積には、明細の右側に **原価計算メモ**が隠れていた(クライアント版 .xlsx には出さない):

```
経費合計（スタッフ謝礼・会場・食材・交通…) / 粗利 = 合計 − 経費 / 利益率 = 粗利 ÷ 合計
例(MTI 新人研修 2022): 合計 1,303,500 / 経費 370,000 / 粗利 815,000 / 利益率 ≈ 62.5%
```

- 見積を起こすときは、この **内部試算を別途 md で添える**と利益率を意識して値付けできる。
- 発行物(雛形 .xlsx)には絶対に載せない。内部試算は案件台帳の作業メモ側に残す。
- MVP では任意。將来 `issue_document.py` に「内部試算シート別出力」を足す余地。

## 注意・既知の差

- **見積と請求で列が違う**: 見積=品名/数量/単価/金額(税抜)、請求=品名/単価/数量/小計/消費税額/税込小計。雛形は別ファイル。
- **酒類(アルコールドリンク)**は外税扱いで消費税を立てない実例があった(請求)。雛形は一律 ×0.1 で計算するので、酒類があれば発行後にガクチョが当該行の消費税を手調整。
- 単価は相場であり拘束しない。案件台帳の実額・見積合意額が常に優先。
