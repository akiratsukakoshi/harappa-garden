# 株式会社フォーシーカンパニー(forc-c.co.jp)

原っぱ大学の toB **紹介元(継続)**。マンション/団地の集会所・公園等のコミュニティイベントを、**複数の物件・案件で継続的に原っぱ大学に紹介・発注**してくれる。**提携契約**を結んでおり、関係そのものが資産(1案件でなく紹介の流れが続く)。

> **登録理由(2026-06-18 ガクチョ)**: 「複数紹介案件をくれるので登録したほうがいい」。単発イベントの集合体だが、**紹介元としての関係が継続**するため企業正本を持つ価値がある。
