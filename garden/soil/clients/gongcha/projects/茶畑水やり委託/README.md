# ゴンチャ 茶畑水やり委託(障がい者雇用のハングアウト)

みうらの森林(京急の場)内にある**ゴンチャの茶畑**(お茶の会社らしく茶の木を栽培)の**水やり・管理を原っぱが受託**する**月次契約**。2025-07 頃から継続(「茶畑水やり委託費のご請求」スレッドが2025-09〜複数)。

> **★お金の流れ(ガクチョ補完)** = **回数単価(都度)** でゴンチャ → 原っぱに請求・入金 → その原資が **障がい者雇用のハングアウト**(障がい者が茶畑の軽作業に従事する場・居場所)の運営に充当される。**原っぱの利益にはならない**(社会的意義の案件)。利益なしのため**金額は管理しない**(finance 上は無視してよい)。

## 障がい者雇用のハングアウト(地続きテーマ)

- 溝口さん(People & Culture/HR)から **2026-05「原っぱ大学での障がい者社員の業務についてご相談」**。茶畑での軽作業・清掃整備・季節作業の補助等を、障がい者社員が従事できる業務として切り出せないか、という相談。
- → **茶畑水やり委託**が、その**受け皿(ハングアウト=居場所/活動の場)**になっている。茶畑の入金がこの運営費に回る構造。
- 障がい者雇用相談そのものは**単独案件化しない**(ガクチョ判断)。この茶畑案件の一部として扱う。
