```
       ② 社員総会イベント(2026-06実施・20万+経費)= みうらの森林を会場に。請求→入金(利益あり)
       ③ 茶畑水やり委託(回数単価・継続)= みうらの森林内のゴンチャ茶畑。入金あるが【利益なし】

       └─ (障がい者雇用相談=②③と地続き。単独案件化はしない)
```
