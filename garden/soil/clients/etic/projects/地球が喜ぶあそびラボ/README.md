# ETIC 地球が喜ぶあそび発見のラボ(ヤマハ発動機協業)(芽)

ETIC. とヤマハ発動機が共同で進めるプロジェクト。ヤマハの「大人の遊び」(バイク/マリン等)の前提=豊かな自然資本を、遊びを通じて**同時再生**する事業の可能性を探る。横浜のオープンイノベーション拠点「**リジェラボ**」を拠点に社外と協働。原っぱ大学には **フィールドワークの共創**と(可能なら)棚卸しフェーズの**アドバイザリー**関与を依頼。

> **「地球が喜ぶ」の定義** = 人間性の再生・コミュニティの再生・自然の再生。今期の ETIC の役割は入口「関係人口を増やす」。
