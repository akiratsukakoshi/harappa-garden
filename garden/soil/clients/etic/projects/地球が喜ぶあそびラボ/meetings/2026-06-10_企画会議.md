# 2026-06-10 ETIC 地球が喜ぶあそびラボ 企画会議(a・客先)

> エティック 倉筋氏・角山氏 + ガクチョ。ヤマハ発動機との協業プロジェクトの 7/9 イベント設計と、その先の三浦半島フィールドワーク構想を整理。**京急との連携提案**で一致。

## プロジェクトの全体像
- ヤマハ発動機(バイク/MTB/マリン=「大人の遊び」)の前提=豊かな自然資本。遊びを通じ自然資本を**同時再生**する事業の可能性を探る。横浜の OI 拠点「**リジェラボ**」で社外協働。
- 「地球が喜ぶ」= 人間性・コミュニティ・自然の再生。今期 ETIC の役割は入口「関係人口を増やす」。
- 今年の三本柱:(1) イベント(理念・事例共有)(2) フィールドワーク(現地体験)(3) 年内棚卸し → 来期接続。
- 原っぱへの依頼 = **フィールドワーク共創** + 可能なら**棚卸しフェーズのアドバイザリー**。

## 7/9 イベント(第2回)
- 第1回(5/16)で出たアイデアを具体企画に育てる回。オープニング=ヤマハ・**福田氏**(リジェラボ統括)。
- パネル:塚越 / ジョン・ミューラー(海山接続の実践者)/ なずな(慶應SFC・19歳・湘南台)+ パラミタ社・大沢氏(尾鷲)。福田氏モデレート → 参加者ブレストで具体化。
- ブレストお題は二案検討中(フィールドワーク構想投げ込み / 登壇者別課題ワーク)。塚越指摘=三浦半島の地形の解像度が低いので「面白いことを」では成果が出にくい → 地形を地図で解説し「ここに行こう」を具体化する運びが有効。

## フィールドワーク構想 + 京急連携(★)
- フィールド = 三浦半島(**野比・走り水**)第一候補。夏〜秋口(**9月目途**)。日帰り/1泊2日の両パターン。10〜15名で濃く。
- 活動案 = 谷筋〜源流〜尾根〜海への沢歩き(1〜3時間)/ 焚き火・自然観察・沼の泥掻き・茶畑水やり等(負荷低め)/ 漁船で沖へ(可否は要確認・慎重)。
- **京急「みうらの森林共創パートナー事業」のフィールド活用を塚越が提案**。横浜で本社至近・コンセプト親和。現共創パートナー=京急・ゴンチャ・三浦観光バス。
- **共創パートナー参画モデル = 年間120万円(10万円/月)をたたき台**(個人会員・BtoBtoC 型の拡張も視野)。
- 効果測定 = なずな氏のバイタル可視化 + 立教大 奇二先生のウェルビーイング指標で**定量化** → 企業の意思決定を後押し(森での定性的変化は明確だが定量指標が不足という課題)。
