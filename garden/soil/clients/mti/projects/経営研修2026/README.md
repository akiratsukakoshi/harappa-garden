## 案件サマリ(打合せから自動蓄積)

- **プログラム**: 当初全5回 → 第4・5回統合で全4回に変更。第4回は6時間(13:00-19:00)+懇親会(延長戦)。会場は新宿ヒルトン地下の貸会議室を仮予約。
- **ゴール**: 主要変革テーマ(事業・組織)を3件程度設定 → 各テーマのアクション・主体者・90日タイムラインを1枚に → 社外取締役へ提示・宣言しフィードバックを取り込む。
- **進め方**: 既製フレームは使わない。現状決算と2030目標のギャップ(数値起点)から入り、MVV と往復。立場別(取締役/執行役員)グループ → 全体共有 → 終盤に輪で合意形成。
- **予算**: **120万円で決済を進めることで合意(06-11)**。
- **社外取締役**: 15:00-19:00 参加予定(州取締役 確定 / 石川氏 返答待ち)。論点深化とレビュー・承認のキーパーソン。
- **★日程(メールで判明)**: 本番WS = **7/23** / 会場下見 = **6/24(水)16:00**(和田同席)/ 次回打合せ = **7/2 or 7/3 9:00**。窓口 = 林 愛也(起案)・中野 絵里奈(実務調整)。
