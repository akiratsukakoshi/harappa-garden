# 2026-05-22 フージャース 新卒・若手育成と宅建方針(a・客先)

フージャース社内の新卒・若手育成会議に塚越参加。**宅建取得率(昨年65%→目標100%)** がボトルネックとなり、内定者期間中の必須化と若手研修の時期再設計を議論。原っぱ大学への業務発注の決定はこの回ではなされず、**6/2 メールで宅建テーマは内製化で進める方針となり若手研修は見送り**(企業 README 参照)。

## 主要論点

### 宅建取得が育成のボトルネック
- 宅建未取得が **8月以降の接客経験を阻む** → 初契約が翌3月頃まで遅延、3-4ヶ月の実地学習機会を失う。
- 営業部だけでなくマンション管理・仕入れ・建築部門のキャリア狭隘化要因にもなる。
- 宅建取得率: 昨年実績 **65%** / 会社目標 **100%**。
- 100% 前提の評価制度は「達成=標準評価」となり、高目標未達リスクや過程評価が見えにくい点を懸念共有。

### 内定者期間中の必須化方針
- 宅建を **内定者期間中に必須化**、未取得者には最終選考辞退を推奨するガイダンスを徹底。
- **6月上旬に内定者向けオンラインキックオフ** 開催、教材を一斉発送し出願状況を会社側で管理。
- **8月中に1泊2日の内定者宅建合宿** を企画。
  - 1日目: 模擬試験・採点・復習 + 夜BBQ
  - 2日目: チームビルディング
  - 候補地: ガリソンの森 / 湘南国際村センター(送迎バス手配可)/ リソルライフフォレスト

### 若手研修(2-3年目)の時期再設計
- 例年は宅建配慮で **11月後ろ倒し** → 全員宅建保有前提で **本来の 5-6月実施** に戻す再設計を検討。
- 育成課題: 3年目までに「見通し力」「可能性の追求」スキル。

### AI 活用方針
- 「使える側になる」方針だが、1年目から依存させると思考力低下懸念 → まず基礎思考フレームワーク優先。

## 数字

- 2025年度新卒: **20名**(配属後3週間順調)
- 2026年度内定者: 現在約 **30名**(6月まで変動可能性)
