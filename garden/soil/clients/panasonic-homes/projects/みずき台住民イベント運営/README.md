# パナHM みずき台住民イベント運営(継続)

神戸みずき台(パナソニックホームズ分譲地)の**住民コミュニティイベントを原っぱ大学が継続運営**する案件。月次でイベントを企画・告知(チラシ)・参加募集・当日運営し、参加実績を報告。四半期ごとに請求。**Plaud 会議録は無く、メールのやり取り(頻繁)で案件が動く** = ドメイン(`panasonic-homes.com`)起点で骨格が立つ横展開の代表例。

> **受注の経緯**: リクルート経由の紹介(コーディネーター=山口)→ パナソニックホームズと直接の運営委託。山口は契約関係のない調整役。

## 案件サマリ(メールから)

- **運営内容** = みずき台の住民向けコミュニティイベントを月次で運営。生き物調査・流しそうめん・季節イベント等。チラシ作成 → 参加募集 → 当日運営 → 参加実績報告 のサイクル。
- **取引形態** = 単発でなく継続運営委託。四半期ごとに請求(年間230万を6月120万・9月110万で分割計上)。
- **窓口** = パナHM 側 5名(大西美帆ほか)+ 外部コーディネーター リクルート山口(契約関係なし)。
- **公的手続き** = 公園使用許可申請等もやり取りに含む(「みずき台東公園 使用許可申請の件」)。
