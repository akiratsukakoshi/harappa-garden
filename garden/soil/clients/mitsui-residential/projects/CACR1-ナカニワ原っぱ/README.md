# 三井 CACR1 ナカニワ原っぱ(中庭コミュニティプログラム)

三井不動産レジデンシャルの分譲マンション **CENTRAL AIR CITY RESIDENCE Ⅰ（CACR1・川崎市幸区塚越4-298-3）** の中庭で、原っぱ大学が月1回の**居住者コミュニティプログラム「ナカニワ原っぱ」**を企画運営する継続案件。チラシ作成 → 掲示 → 当日開催 → 活動レポート、という**月次サイクル**が定常化している。**パナHM みずき台と同型(継続運営型)**。

> 6/15 京急品川会議でガクチョが実績紹介した「川崎のマンション(三井不動産)・人工芝の中庭・月1回の季節遊び・約250戸/参加率10%弱」がこの案件。

## 月次サイクル(メールから)

```
チラシ案 作成 → 管理(奈良)へ確認・掲示依頼 → 当日開催(天候で実施可否連絡)→ 活動レポート/報告書 提出
```
- 月1回開催(土日中心)。例:5/30(土) CACR1、9/27、10/25 等。
- 雨天時は「予定通り実施します/中止」を前日連絡。
- 年次:総会用レポート(2026-04「総会用ナカニワはらっぱレポート」)= 管理組合総会向けの年間まとめ。
