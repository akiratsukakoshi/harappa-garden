# 三井不動産レジデンシャル株式会社（MFR）

原っぱ大学の toB クライアント。三井不動産の分譲マンション事業会社。川崎市幸区塚越の物件 **CENTRAL AIR CITY RESIDENCE Ⅰ（CACR1）** の中庭を舞台に、原っぱ大学が月1回の**中庭コミュニティプログラム「ナカニワ原っぱ」**を企画運営する、**単一物件・継続運営型**(パナHM みずき台と同型)。6/15 京急品川会議でガクチョが実績として挙げた「川崎のマンション・人工芝中庭・月1回季節遊び・約250戸/参加率10%弱」がこの案件。

> **物件**: CENTRAL AIR CITY RESIDENCE Ⅰ（CACR1）/ 川崎市幸区塚越4-298-3 / 通称「塚越北」「ナカニワ原っぱ」
> ⚠️ ガクチョが会議で言及した「川崎・夜光(やこう)のマンション」と同一物件か要確認(メール上の表記は「塚越北/CACR1」)。

## 横断メモ

- **当たっているテーマ** = 分譲マンションの**居住者コミュニティ醸成**。中庭(ナカニワ)を人が集まる場にする月次プログラム。パナHM みずき台・京急 桐畑と同じ「継続運営型」。
- **お金の向き** = 原っぱ → 三井(入金=売上)。**前受金で全額入金済み** → 約11万/月で前受を月次取り崩して売上計上(現契約 〜2026-09末)。**請求先が事業主 MFR か管理会社 MFRS か要確認**。
- **デベロッパー横展開の母体** = 三井(MFR)は大手。CACR1 の実績が他物件に広がる余地。原っぱの「デベロッパー協業コミュニティ運営」の代表実績の一つ。
