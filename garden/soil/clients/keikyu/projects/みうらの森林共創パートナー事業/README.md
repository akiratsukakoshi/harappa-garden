# 京急 みうらの森林共創パートナー事業(継続・第2期)

横須賀市野比の京急所有地「**みうらの森林**」(横須賀市野比4丁目17番1号)を舞台に、原っぱ大学が**森林共創パートナー事務局**として運営する継続プラットフォーム事業。京急(主管=**広報部**・佐野広報課長 / キーパーソン=新しい価値共創室の杉山)が土地提供・主管。**複数のパートナー企業が月額を払ってジョイン**し、月1回の活動(オープニングデー/ワークデー)を回す。事業サイト https://miuranomori.keikyu.co.jp/

> **共創パートナー型の妙**: 1つの「場」が複数企業の接点になり、そこから派生案件(桐畑・品川・ゴンチャ直接案件)が生まれる。みうらの森林は京急との関係の**旗艦**。

## 事業の仕組み(メール+ガクチョ補完)

- **場** = みうらの森林(京急所有・横須賀市野比)。京急が土地と社内調整・主管。
- **運営** = 原っぱ大学が森林共創パートナー事務局。月1回活動(オープニングデー=お披露目/体験、ワークデー=整備作業)。
- **収益** = パートナー企業が月額を払って活動を支える。現在2社で**各11万/月 × 2社 = 計22万/月(264万/年)**:
  - **ゴンチャジャパン**(11万/月・年額一括・支払済)
  - **三浦観光バス**(11万/月・3ヶ月毎・次回6月末に打診)
- **連携の広がり** = 三浦観光バス(PR・生物調査・現地活動)/ ヤマハ(協業相談)/ 森未来(オープンデイ)。
- **発展** = 横須賀市野比で**自然共生サイト**の取得を、**立教大学の奇二(きじ)先生**(kiji@ka2.so-net.ne.jp)と推進。**取得は来年度以降・今年は下地調査**(生物調査=2026-05-19 の「生物調査について」メールはこの一環)。

## 毎月のお金の流れ(共創パートナー型の特異点・ガクチョ補完)

通常のクライアント案件と**逆向きの支払い**を含む。京急は「請求先」でなく**共同事業者(=原っぱから見て支払先)**:

```
① パートナー2社(ゴンチャ・三浦観光バス) ──各11万/月──▶ 原っぱ大学(売上=計22万/月)
② 売上22万 − 原価(経費)= 粗利
③ 京急への支払い = 粗利 × 30%(京急の利益取り分)+ 水道光熱費
④ 前月分を共有レポートで 原っぱ ──共有──▶ 京急(月次集計)
⑤ 京急が請求書を発行           京急 ──請求──▶ 原っぱ
⑥ 原っぱが支払い(外注費)       原っぱ ──支払──▶ 京急(③の金額)
   → 原っぱの手残り = 粗利 − ③
```

**2026年5月の実額([共有レポート](https://docs.google.com/spreadsheets/d/19IWqq-DcBtM8-rOuXxX3xRrqmUzjhcnh5ycjRERtEIY/edit) 202605 タブ)**:

| 項目 | 金額 |
|---|--:|
| 売上(ゴンチャ11万 + 三浦観光バス11万) | 220,000 |
| 原価(水光熱・保険・人件費・弁当・耕運機・材料・ピートモス・氷虫よけ) | △93,271 |
| **粗利** | **126,729** |
| 京急の利益取り分(粗利 × 30%) | 38,019 |
| 水道光熱費 | 10,000 |
| **→ 京急への支払い合計(外注費)** | **48,019** |
| 原っぱの手残り(粗利 − 京急支払い) | 78,710 |

- **入金元** = パートナー企業(売上 22万/月)/ **支払先** = 京急(外注費=粗利30%+水道光熱費、月変動)。→ freee 上、京急取引先は**支払先**として扱う。
- **案分比率 = 粗利の30%が京急取り分**(残り70%が原っぱ。ただし水道光熱費は京急への支払いに加算)。
- **毎月の共有レポート** = [パートナー事業レポート(Google Sheets)](https://docs.google.com/spreadsheets/d/19IWqq-DcBtM8-rOuXxX3xRrqmUzjhcnh5ycjRERtEIY/edit)(月タブ=`YYYYMM`。原っぱが作成 → 京急共有 → 京急が請求発行)。
- finance 突合(Mode A/D)では **①パートナー入金(売上22万)** と **⑥京急への外注費支払い** の両方を見る。
