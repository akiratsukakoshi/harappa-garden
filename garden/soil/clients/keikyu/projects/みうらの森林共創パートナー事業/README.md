# 京急 みうらの森林共創パートナー事業(継続・第2期)

横須賀市野比の京急所有地「**みうらの森林**」(横須賀市野比4丁目17番1号)を舞台に、原っぱ大学が**森林共創パートナー事務局**として運営する継続プラットフォーム事業。京急(新しい価値共創室)が土地提供・主管。**複数のパートナー企業が月額を払ってジョイン**し、月1回の活動(オープニングデー/ワークデー)を回す。事業サイト https://miuranomori.keikyu.co.jp/

> **共創パートナー型の妙**: 1つの「場」が複数企業の接点になり、そこから派生案件(桐畑・品川・ゴンチャ直接案件)が生まれる。みうらの森林は京急との関係の**旗艦**。

## 事業の仕組み(メール+ガクチョ補完)

- **場** = みうらの森林(京急所有・横須賀市野比)。京急が土地と社内調整・主管。
- **運営** = 原っぱ大学が森林共創パートナー事務局。月1回活動(オープニングデー=お披露目/体験、ワークデー=整備作業)。
- **収益** = パートナー企業が月額を払って活動を支える。現在2社で**計11万/月(132万/年)**:
  - **ゴンチャジャパン**(年額一括・支払済)
  - **三浦観光バス**(3ヶ月毎・次回6月末に打診)
- **連携の広がり** = 三浦観光バス(PR・生物調査・現地活動)/ ヤマハ(協業相談)/ 森未来(オープンデイ)。
- **発展** = 横須賀市野比で**自然共生サイト**の取得に向けて活動中。

## 毎月のお金の流れ(共創パートナー型の特異点・ガクチョ補完)

通常のクライアント案件と**逆向きの支払い**を含む。京急は「請求先」でなく**共同事業者(=原っぱから見て支払先)**:

```
① パートナー企業(ゴンチャ・三浦観光バス) ──月額──▶ 原っぱ大学(入金=売上)
② 売上 − 経費 − 原価 = 利益 → 原っぱ / 京急 で案分
③ 前月分を共有レポートで 原っぱ ──共有──▶ 京急
④ 京急が請求書を発行           京急 ──請求──▶ 原っぱ
⑤ 原っぱが支払い(外注費)       原っぱ ──支払──▶ 京急(案分した京急取り分)
```

- **入金元** = パートナー企業(売上)/ **支払先** = 京急(外注費)。→ freee 上、京急取引先は**支払先**として扱う。
- **毎月の共有レポート** = [前月分パートナー事業レポート(Google Sheets)](https://docs.google.com/spreadsheets/d/19IWqq-DcBtM8-rOuXxX3xRrqmUzjhcnh5ycjRERtEIY/edit)(原っぱ→京急共有)。
- finance 突合(Mode A/D)では **①パートナー入金** と **⑤京急への外注費支払い** の両方を見る。
