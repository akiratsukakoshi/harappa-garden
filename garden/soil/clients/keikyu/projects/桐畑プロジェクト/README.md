# 京急 桐畑プロジェクト(プライム横浜 コミュニティイベント企画運営)

みうらの森林の実績から**派生**した、京急の新規分譲物件「**プライム横浜**」(横浜市神奈川区桐畑)の居住者コミュニティ形成を、原っぱ大学が企画運営する案件。**パナHM みずき台と同型のコミュニティ運営委託**。委託契約書を締結済(2025-12-19 最終案)。

> **受注の経緯**: みうらの森林共創パートナー事業の実績 → 京急社内で評価 → 新規物件のコミュニティ運営を直接発注。施設は「みうらの森林(横須賀市野比)」を優先使用できる(契約書 第7条)= 旗艦案件からの派生が契約条項にも表れている。

## 契約要点(委託契約書 251219 最終案)

- **委託者(甲)** = 京浜急行電鉄株式会社(取締役社長 川俣幸宏)/ **受託者(乙)** = HARAPPA株式会社(代表取締役 塚越暁)
- **物件(本物件 = プライム横浜)**:
  - プライム横浜 ヒルズ(横浜市神奈川区桐畑2番・170戸・地上9階・竣工2027/2・引渡2027/3)
  - プライム横浜 ヴィラ(桐畑3-5番・29戸・地上10階地下1階・竣工2027/5・引渡2027/6)
- **業務** = 居住者のコミュニティ形成促進(イベント企画・設計・実施)。管理組合(丙)が調整・広報・参加受付、原っぱが企画・運営・当日進行・報告書。
- **委託期間** = 契約締結日 〜 **2029年3月末日**(完了後、丙が継続委託する場合は別契約)。
- **業務委託料 = 税抜150万円**(契約書 第13条・別表):

| 項目 | 金額(税抜) | 実施月(予定) | 支払期日(予定) |
|---|--:|---|---|
| (1) コミュニティ活動実施計画の策定 | 70万円 | 2027-01 | 本契約締結月の翌月末日 |
| (2) イベント実施 2回 | 40万円 | 2027-04〜 | 2027-05 末 |
| (3) イベント実施 2回 | 40万円 | 2028-04〜 | 2028-05 末 |

- **支払** = 乙指定口座へ振込(手数料 甲負担)。支払期日の前月末までに請求。
- **施設優先使用**(第7条) = みうらの森林の敷地・共用施設・什器を無償優先使用可。
- 守秘義務・個人情報保護あり(居住者個人情報を扱う)。管轄 = 東京地裁。
