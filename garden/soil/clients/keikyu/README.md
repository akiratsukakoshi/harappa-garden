# 京浜急行電鉄株式会社(京急)

## 横断メモ

- **当たっているテーマ** = 京急の不動産/沿線エリアにおける**住民・地域コミュニティ醸成**。みうらの森林を旗艦に、新規物件(桐畑=プライム横浜)・地域(三浦海岸)・再開発(品川)へ横展開。
- **共創パートナー型の妙** = 原っぱ大学が「場の事務局」を担い、複数企業から月額を集めて場を回す。京急は土地と主管。**1つの場が複数案件・複数企業の接点になり、そこから派生案件が生まれる**(桐畑・品川は京急の別部署/物件発)。
- **受注因果** = みうらの森林の実績 → 京急社内で評価 → 新規物件のコミュニティ運営(桐畑)を直接発注、という連鎖。パートナー企業(ゴンチャ)からは別の直接案件(社員総会)も派生。
