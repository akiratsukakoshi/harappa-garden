# 株式会社エイチ・ティー(HT / hxt.jp)

原っぱ大学の toB **委託元(紹介商流のハブ)**。エイチ・ティーが顧客企業に研修を提案し、**体験型研修の実施を原っぱ大学に委託**する。原っぱから見た**契約相手・入金元はエイチ・ティー**(委託費)で、**研修の実施先(エンドクライアント)は別企業**(白井松新薬・一丸ファルコス)という、これまでにない **代理店/紹介商流型(間接商流)** の構造。

> **整理方針(2026-06-18 ガクチョ確定・案A)**: お金を辿り、**エイチ・ティーを企業フォルダの「家」**にする(京急で「支払先」を家にしたのと同型の発想)。エンドクライアント(白井松/一丸)は **案件(project)** として家の中に置き、各案件 frontmatter の `end_client` で実施先を明記する。

## 商流の構造(間接商流)

```
                  研修を提案
エイチ・ティー(hxt.jp) ───────────→ エンドクライアント(白井松新薬 / 一丸ファルコス)
   │  照井直哉・鳥羽瀬建(KEN)            戌亥・井上(白井松) / 安藤(一丸)
   │
   │ 実施を委託(発注書)
   ▼
原っぱ大学 ──(委託費を請求)──→ エイチ・ティー が支払い
   研修を現地実施
```

- **白井松とエンドの安藤(一丸)が同席**: 白井松チームビルディング研修(5/11-12)の当日コーディネートに、親会社・一丸ファルコスの安藤さんが参加(チーム分け作成)。**白井松と一丸は取締役同一・親子会社**で関係が地続き(→ 一丸への横展開提案につながった)。
