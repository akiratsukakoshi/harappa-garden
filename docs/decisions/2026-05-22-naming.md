# 2026-05-22 — プロジェクト名: HARAPPA Management Garden (HMG)

## 決定

新プロジェクト名を **HARAPPA Management Garden (HMG)** とする。略称 HMG。

## コンテキスト

HMC (HARAPPA Management Cockpit) の進化版として、AI中心の業務フローを実現するプロジェクトを立ち上げる必要があった。HMCの「Cockpit」は **「人間=中心、ツール=周辺、計器類=モニタリング対象」** という空間配置を一語で表現していた。新プロジェクトは **その空間配置自体を反転または再構成する** ことが目的。

## 検討した候補

ネーミング比喩として 6 つの世界観を比較:

| 案 | 一語 | 評価 |
|---|---|---|
| **学園 (Gakuen)** | ガクチョー / 教務 | HARAPPA大学・ガクチョーと整合。だがAI中心のニュアンスが弱い |
| **商家 (Shōka)** | 番頭 / 蔵 | 構造は完璧。だが塚越さんに馴染みが薄い |
| **工房 (Atelier)** | 親方 / 棟梁 | 創造的だが事務系業務との距離 |
| **邸宅 (Estate)** | 執事 / Butler | 階級制の匂い、HARAPPAの土の匂いと乖離 |
| **宮廷 (Court)** | 王 / 宰相 | 仰々しい、情緒的接続が薄い |
| **編集部** | 編集長 / デスク | 現代的だがHARAPPAとの距離 |
| **🟢 Garden (庭・原っぱ)** | 庭師 / 区画 | HARAPPA(=自然)直結、「育てるシステム」と一致、Cockpit→Gardenの構造変化が一語で表現できる |

## 決定の根拠

**Harappa Management Garden** を選んだ理由:

1. **構造変化の象徴性** — Cockpit(機械を操る)→Garden(育てる生態系)で、人間=操縦者→人間=庭師、ツール=計器→エージェント=生態系、という根本転換を一語で表現できる
2. **ブランド整合** — 「原っぱ」(=自然のフィールド)と Garden が二重写しになり、事業ブランドと連続する
3. **語彙の豊富さ** — 庭師・区画・種・土壌・剪定・収穫・番人・寄合・天気・季節・苗床・蔵など、業務設計の語彙を派生できる
4. **「育てる」コンセプトとの一致** — HMC時代から CLAUDE.md にあった「育てるシステム」と本質的に同じ
5. **頭文字継承** — HMC→HMG で「HM_」を継承し、進化の連続性を示せる
6. **塚越さんが「腑に落ちる」と評価** — 議論の中で「凄くイメージがわく」「方向修正として優秀」「原っぱの土のイメージをとらえている」と言及

## 代替案がなぜ落ちたか

- **学園**: ガクチョー呼称と整合する魅力はあったが、**「AIをパートナーにAIセンターな業務フローを作っている」というニュアンスがゼロ**になる(塚越さんからの指摘)。「ガクチョーが学園を運営する」は HMC とほぼ同じ構造を表してしまう
- **商家・工房・邸宅・宮廷・編集部**: 馴染みの薄さ・HARAPPAブランドとの距離・階級的含意などの理由で却下

## 影響範囲

この決定は以下のすべてを規定する:
