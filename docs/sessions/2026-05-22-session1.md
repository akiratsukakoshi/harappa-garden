## 今日決めたこと

### 1. 土壌の設計テンション5つの合意

| | テンション | 合意 |
|---|---|---|
| (1) | 範囲 | マスター系=ローカル統一、ドキュメント=外部+インデックス のハイブリッド |
| (2) | 形式 | **Karpathy LLM Wiki 方式**(markdown中心 / frontmatter で構造化 / [[link]]) |
| (3) | アクセス層 | MCP で統一(最初は read のみ) |
| (4) | 最初の苗床 | スタッフマスター |
| (5) | 編集権限 | 庭師 + Claude 双方。log.md 必須、判断系編集は剪定待ち |

**(2) は転換点**: 当初 Claude 推奨は SQLite だったが、塚越さんから Karpathy の LLM Wiki が提案され、転換。理由: 「Claude の知の構造が見えやすく、塚越さんが編集しやすい」。Garden 哲学との整合も高い(育てる/見回る/剪定する)。

- **role 1軸**: 代表 | 運営 | 業務委託 | アルバイト(雇用区分とは別軸)

- frontmatter: `type / status / role / kana / freee_id / freee_type / email / nicknames / joined / linked_* / sources / private_details / last_updated`
