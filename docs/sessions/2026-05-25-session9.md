### 重要な技術的発見

- **ssh 非対話シェル / cron では `~/.bashrc` が読まれない** → ランチャーは **フルパス必須**(`~/.npm-global/bin/claude`)
- TZ = JST なので cron 式そのまま種スキーマと整合
- `/opt/` は root 所有 → `/opt/garden/` ではなく `~/garden/`(secret ADR と同じホーム範囲)で運用するのが楽
- 5984(CouchDB デフォルト)は空き、ただし既存パターン通り 127.0.0.1 バインド + Nginx Proxy Manager 経由公開が整合的
