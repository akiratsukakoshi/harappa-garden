## 学び・メモ

- **「実運用の細かい指示」こそ SKILL の栄養**: 今日ガクチョと往復した粒度(過去分除外・表記ゆれ・色の意味)を SKILL に書き戻したことで、次回から Discord ガクコで完結する。実演 → 蓄積のループが区画を育てる
- `process.exit()` は finally を実行しない(Node.js)。lock 解放のような後始末は exit hook に置く。**「失敗するとガードが残って次回が黙死する」型**は S42 の Triage 欠落と同じ「静かに欠落する」族 — 失敗パスの観測性を設計時に疑うこと
- Sheets の書き込みは型を意識する(str() 一律 → `'` つき文字列セル)。プルダウン選択肢は既存セル値と**表記完全一致**(全角括弧まで)させないと無効マークが出る
- Drive API の move 直後の list は整合性遅延で不正確なことがある(数秒置いて再取得)
- launcher テストの日付は `toISOString()`(UTC)でなくローカル日付で(JST 0-9 時に落ちるフレーク)
